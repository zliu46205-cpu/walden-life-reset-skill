# 安装说明

## 1. 下载

点击 GitHub 页面右上角 `Code`，选择 `Download ZIP`，解压。

## 2. 复制 skill 文件夹

复制仓库里的这个文件夹：

```text
walden-life-reset
```

到 Codex skills 目录。

Windows:

```text
C:\Users\<你的用户名>\.codex\skills\
```

macOS / Linux:

```text
~/.codex/skills/
```

## 3. 重启 Codex

重启后，在对话中测试：

```text
使用 $walden-life-reset，帮我做一次生活心态整理。
```
